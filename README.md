# üíª clcltr: A Simple Web Calculator

A clean, responsive, and functional calculator implemented using modern web technologies (HTML, CSS, and JavaScript). This project provides a basic but essential tool for performing quick arithmetic calculations directly in the browser.

## ‚ú® Features

The `clcltr` application focuses on simplicity and ease of use, providing the fundamental tools necessary for day-to-day calculations.

* **Basic Arithmetic:** Supports standard operations: Addition (`+`), Subtraction (`-`), Multiplication (`*`), and Division (`/`).
* **Square Root Functionality:** Includes a button for calculating the square root of a number ($\sqrt{x}$).
* **Clear Functions:** A "Clear Entry" (CE) button to erase the current input, and an "All Clear" (AC) button to reset the entire calculation state.
* **Decimal Support:** Allows calculations involving non-integer values.
* **Responsive Design:** Optimized layout for use on both desktop and mobile devices.

## üöÄ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You only need a modern web browser (e.g., Chrome, Firefox, Edge) to run this application.

### Installation (Local Setup)

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/caawq/clcltr.git](https://github.com/caawq/clcltr.git)
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd clcltr
    ```
3.  **Open the application:**
    Simply open the `index.html` file in your preferred web browser.

    ```bash
    open index.html
    # OR (on Windows)
    start index.html
    ```

## üí° Usage

The calculator is straightforward to use:

1.  Click the number buttons to input your desired value.
2.  Click an operator button (`+`, `-`, `*`, `/`) to select the operation.
3.  Enter the next number.
4.  Click the equals (`=`) button to see the result.
5.  Use the $\sqrt{x}$ button to calculate the square root of the displayed number.
6.  Use **AC** to clear the entire calculation and display.
7.  Use **CE** to clear the currently entered number.
